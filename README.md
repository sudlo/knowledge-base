## ⚡ Overview

**IronHide** (formerly Knowledge Base) is a standalone, lightweight Windows executable designed to be the ultimate daily resource for cloud and infrastructure professionals.

It combines a massive offline database of cloud services, architectural patterns, and security frameworks with a **completely free, keyless AI assistant** capable of writing code, analyzing logs, and parsing CSV ticket dumps.

**No API keys. No Python installation. No bloated dependencies. Just double-click and go.**

-----

## 🔥 Core Features

### 🧠 Massive Offline Knowledge Base

  - **250+ Cloud Services:** Comprehensive overviews and deep dives across **AWS, Azure, and Google Cloud**.
  - **Dynamic Routing:** Instantly routes you to official, up-to-date Microsoft, AWS, or GCP documentation.
  - **Cross-Platform:** Deep dives into Windows Server (AD, FSMO, DNS) and Linux (systemd, eBPF, kernel tuning).
  - **Virtualization:** VMware vSphere, ESXi, vMotion, and DRS troubleshooting.

### 🏛️ Architecture & Security

  - **System Design:** Caching strategies, HA/DR, Microservices (Sagas/CQRS), and the Well-Architected Framework.
  - **Security Posture:** Zero Trust, IAM (SAML/OIDC/OAuth), Cryptography (mTLS), and AppSec/SIEM overviews.

### 📋 Log & Ticket Analyzer

Don't stare at massive error walls. Drop your data directly into the app for instant root-cause analysis:

  - **Supported Formats:** Paste raw text, or upload `.log`, `.txt`, and **`.csv` ticket dumps**.
  - **Trend Analysis:** Identifies recurring failures and systemic fragility across Jira/ServiceNow ticket exports.
  - **Actionable Output:** AI strictly formats responses into *Root Cause*, *Fixes*, and *Preventive Measures*.

### 🤖 IronHide AI Coder

A built-in, polyglot AI assistant strictly engineered for DevOps and Infrastructure tasks.

  - **Supported Languages:** Python, Bash, PowerShell, Terraform, Ansible, Go, and more.
  - **Sourced Code:** IronHide is instructed to provide official GitHub or Registry links for the modules it generates.
  - **Serverless & Free:** Powered by a free AI routing endpoint. No "Bring Your Own Key" (BYOK) required.

## 🚀 Getting Started

Because IronHide is compiled via a streamlined GitHub Actions pipeline, getting started takes literally seconds.

1.  Navigate to the **Releases** tab on the right side of this repository.
2.  Download the latest `Knowledge Base.zip` artifact.
3.  Extract the folder and double-click `Knowledge Base.exe`.
4.  *That's it.*

*(Note: Windows SmartScreen may flag the `.exe` because it is not signed with an expensive developer certificate. Click "More Info" -\> "Run Anyway".)*
